# Flutter-smart-home-task
Smart Home MQTT App

This is a flutter-based smart home application that uses MQTT protocol to control devices like lights, fans and Ac.

--> Features 
Connect to mQTT broker
Turn ON/OFF devices
Real-time updates 
Simple UI
